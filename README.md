### Installation

_Below is the way to run this project on your device. Please star this repo first :)_

1. Star this repo :)
2. Clone the repo
   ```sh
   git clone https://github.com/Muhthoriqas/My-Portofolio.git
   ```
3. Go to project
   ```sh
   cd My-Portofolio
   ```
4. Open Your Text Editor
   ```sh
   code . (for shorcut to open VScode)
   ```
5. Install NPM
   ```sh
   npm install
   ```
6. Run Local Server
   ```sh
   npm start
   ```
